# Commercial intent phrase packs

Deterministic, editable phrase families for CTA/action interpretation. Not runtime NLP.

- `action-taxonomy.json` — action families, stages, friction, verbs/nouns/phrases, negative context
- `multilingual-phrases.json` — per-locale synonyms for core families (v1: en, es, fr, de, it, pt, nl)
- `blocker-taxonomy.json` — objection/trust categories and path/phrase hints
- `page-role-taxonomy.json` — URL/title hints for page roles

Privacy: packs classify **labels and paths only** — never form values or search queries.
