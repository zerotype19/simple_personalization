Optiview hosted snippet — install examples

Production CDN (recommended)

<script async src="https://cdn.optiview.ai/si.js" data-si-key="YOUR_PUBLIC_SNIPPET_KEY"></script>

Verify in the console

typeof window.SessionIntel === "object"
typeof window.SessionIntel.getExperienceDecisionEnvelope === "function"
window.SessionIntel.getExperienceDecisionEnvelope()

Subscribe to one surface

const surfaceId = "article_inline_mid";
window.SessionIntel.subscribeToDecision(surfaceId, () => {
  const decision = window.SessionIntel.getExperienceDecision(surfaceId);
});

Self-host (rebuild required for custom API/CSS hosts)

Upload cdn/si.js and cdn/si-inspector.css together. A prebuilt bundle still points at the baked worker_url and snippet_origin in version.json unless you rebuild with VITE_SI_WORKER_URL and VITE_SI_SNIPPET_ORIGIN. 

<script async src="https://YOUR_HOST/si.js" data-si-key="YOUR_PUBLIC_SNIPPET_KEY"></script>

See README.txt and docs/CUSTOMER_INSTALL.md in this bundle.
